Install Python and MEGACC and make sure to select "Add to PATH" during installation.

The order of installation is not important.

After installation, check if the installations are successful:

  Press Windows + R and type cmd

  A command window will open:
  Check the python installation with the command: python --version
  The command should work and give you the version of python

  Check the installation of MEGACC with the command: megacc --version
  The command should work. Otherwise restart and try again. If it still doesn't work
  no, it's because you didn't select "add to path" during installation.

If everything works, run the "execute once python is installed" file to install the libraries required by the script.
Or install the following libraries manually :
  pandas
  biopython
  python-docx
  openpyxl

Make a copy of the src and file folders on your computer. The BLASTparse script is in the /src/ folder.
The /files/ folder is necessary for the script to work properly (contains MUSCLE and phylogenetic tree instructions).
